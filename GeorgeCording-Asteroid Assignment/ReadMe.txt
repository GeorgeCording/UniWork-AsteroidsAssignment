--------------------------------------------------------------
This is the read me file of the asteroids game, created by George Cording
This project was created in IntelliJ and may require a Java Development Kit to run
--------------------------------------------------------------
To run the program, call the public static void main function in the Game Class, located in the core folder.
This will initiate the game.
--------------------------------------------------------------
Controls for the game:
Arrow Keys: Movement
Space bar: Shoot
R key: Restart
C key: Toggle Chaos mode (requires restarting the game wit the R key)
I key: Toggle Image key (Also requires starting a new game with the R key)
Numbers 1-6: Select player ship.
